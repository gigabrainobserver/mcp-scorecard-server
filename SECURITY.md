# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in MCP Scorecard Server, please report it responsibly.

**Email:** security@mcp-scorecard.ai

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and aim to provide a fix within 7 days for critical issues.

## Scope

This policy covers the MCP Scorecard Server package and its interactions with the MCP Scorecard API.
